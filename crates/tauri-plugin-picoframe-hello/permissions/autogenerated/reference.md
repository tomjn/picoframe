## Default Permission

Allows the hello plugin's commands.

#### This default permission set includes the following:

- `allow-hello-greet`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`picoframe-hello:allow-hello-greet`

</td>
<td>

Enables the hello_greet command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`picoframe-hello:deny-hello-greet`

</td>
<td>

Denies the hello_greet command without any pre-configured scope.

</td>
</tr>
</table>
